#  Women Safety Rating System

A web application designed to improve women's safety by allowing users to rate locations, view safety scores, and send emergency SOS alerts.

##  Features
- User Registration & Login
- Location-based Safety Score
- Feedback System
- SOS Emergency Alert
- Real-time Data Storage

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express.js
- MySQL
